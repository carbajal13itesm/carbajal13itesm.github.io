What is iterative design? Explain its importance in interaction design.

Diseno de iteracion es simplemente pasar de una version de diseno a otra mejor, y en cada paso, se realiza una evaluacion de usabilidad para certificar la mejora. Cada iteración conlleva la eleccion de un pequeno conjunto de requisitos que permite rapidamente disenar, implementar y probar.

Reduce los riesgos de forma temprana. El progreso es visible desde las primeras etapas. Refina el sistema a las necesidades reales de los usuarios. Gestiona la complejidad: el equipo no se ve abrumado por la «parálisis del analisis». El conocimiento adquirido en un ciclo se puede utilizar metódicamente para mejorar el propio proceso de desarrollo.
